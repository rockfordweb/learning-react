React Basics

React.render

accepts a ReactElement

accepts a DOMElement

optionally accepts a callback

React.createElement

accepts an html tag name as a string, or a ReactClass

takes properties object to use with html or ReactClass element

all other arguments are passed in as children

returns a new ReactElement

React.createClass

accepts an object that requires a render method

returns a new ReactClass
